# Arduino_UI_Project
**Sadece ana kaynak koduna hızlıca göz atmak veya kopyalamak isterseniz main_code.txt dosyasını indirebilirsiniz. Projenin tam içeriğine erişmek için ise hazırlanan .rar dosyasını kullanabilirsiniz. Bu arşiv içerisinde; I2C adres tespit yazılımı, bağlantı şemaları, gerekli OLED kütüphanesi ve ana yazılımın .ino formatındaki dosyası yer almaktadır.
